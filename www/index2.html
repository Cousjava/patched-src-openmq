Open Message Queue

Open Message Queue (Open MQ) is a product that implements and facilitates message transfer between applications. The Open Message Queue product is the community version of Sun Java System Message Queue.

Open MQ is a complete implementation of Java Message Service (JMS). This provides a convenient way to connect your applications together so that they can run independently from each other. Open MQ can be used as a buffer between applications that can produce and consume messages and different rates. Open MQ can also distribute messages in a prescribed way. This is flexible and can be changed without impacting the applications that are sending or receiving messages. Open MQ implements a variety of message delivery patterns that allow it to be customized to meet a wide variety of requirements involving speed, buffer storage, resiliency, and stability and independence.
