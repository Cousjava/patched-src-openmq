2. Project Summary
    2.1. Project Description:

    JAAS allows multiple LoginModules to be chained together so that more than
    one authentication technology can participate in the authentication process.

    This feature is to provide JAAS LoginModules for the MQ built-in Ldap and
    File-based authentication so that they can be used as LoginModules either
    by themself or  chained with customer LoginModule(s) to participate JAAS
    authentication process when MQ "jaas" authentication type is enabled.
