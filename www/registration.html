If the user registers the product when they perform the installation, this command is not required. However, if they choose not to register at installation time, but want to register later, this provides that, without requiring a re-install.

New installer 'register only' mode

If the user decides to skip registration, he or she can perform registration after installation by running the installer in a 'register only' mode. In this mode, the installer will display only the screens related to registration. A requirement here is that MQ 4.2 needs to be already installed - since registration will require the local service tag registry.

The MQ 4.2 'register only' mode is activated by running the installer with a new '-r' option. Relevant usage help output in bold below.
